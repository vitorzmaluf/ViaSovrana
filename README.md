# Via Sovrana Landing Page

Landing page estática para a transportadora Via Sovrana.

## Arquivos

- `index.html`: estrutura da página
- `styles.css`: identidade visual, responsividade e layout
- `script.js`: menu mobile, destaque de seção e envio do formulário para WhatsApp

## Como usar

1. Descompacte o arquivo `.zip`
2. Abra o arquivo `index.html` no navegador
3. Para publicar, envie os três arquivos para sua hospedagem

## O que você precisa ajustar antes de publicar

1. Trocar o número do WhatsApp em `index.html` e `script.js`
   - Procure por: `5500000000000`
2. Trocar o e-mail no rodapé
   - Procure por: `contato@viasovrana.com.br`
3. Inserir CNPJ/endereço quando estiver definido
4. Substituir a ilustração do caminhão por uma imagem real ou mockup da frota, se quiser

## Cores usadas

- Azul marinho: #1B3A6B
- Laranja: #FF6B1A
- Branco: #FFFFFF
